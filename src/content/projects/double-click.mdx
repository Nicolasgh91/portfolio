Double Click™ es un sistema de gestión propietario que reduce la dependencia de múltiples SaaS. El foco actual del desarrollo es el **módulo de gestión de stock**; el ecosistema ya orienta reservas, facturación y retención de clientes para el estudio de Pilates.

## Cuándo tiene sentido reemplazar SaaS

Tiene sentido construir un sistema propio cuando la empresa lleva más de 18 meses pagando suscripciones que suman más de USD 500/mes, cuando existen procesos críticos que ninguna herramienta existente resuelve bien, o cuando el equipo ejecuta tareas manuales repetitivas que podrían automatizarse.

En el caso Double Click™, el estudio pagaba tres herramientas independientes que no se comunicaban entre sí. La dueña invertía dos horas diarias en consolidar información manualmente. El sistema propio centralizó las tres funciones y eliminó las tres suscripciones — con amortización del desarrollo en menos de un año.

## Stack técnico

- **Frontend**: Next.js 14 + Tailwind CSS
- **Backend**: Python / FastAPI
- **Base de datos**: PostgreSQL
- **Integración**: Pasarelas de pago locales (Mercado Pago)
