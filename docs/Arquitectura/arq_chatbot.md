# Chatbot — Documentación técnica

## Arquitectura implementada

El chatbot está integrado en el sitio como un widget flotante de costo operativo cero. El widget (`public/chatbot/widget/chatbot.html`) se carga como iframe desde `src/layouts/Layout.astro`, lo que lo hace presente en todas las páginas del sitio sin modificar su estructura. El contenido que alimenta al bot — servicios, artículos y datos de contacto — vive en archivos JSON estáticos dentro de `public/chatbot/data/`, que se inyectan como contexto en cada llamada a la API. La comunicación con Gemini 2.0 Flash no ocurre directamente desde el browser sino a través de una Vercel Edge Function (`api/chat.js`), que actúa como proxy seguro: recibe el mensaje del usuario, construye la llamada a Gemini usando la API key almacenada como variable de entorno en Vercel, y devuelve la respuesta. De esta forma la key nunca queda expuesta en el código fuente ni en las herramientas de desarrollo del browser. Para actualizar el conocimiento del bot basta con editar los JSON y hacer redeploy; para replicar el chatbot en el sitio de un cliente nuevo, se copian los archivos de `public/chatbot/`, se actualizan los JSON con el contenido del cliente, y se crea una Edge Function nueva con su propia API key de Gemini — obtenible de forma gratuita desde Google AI Studio sin tarjeta de crédito.
